# Windows Server Rollen

Windows Server Rollen sind vordefinierte Funktionen, die einem Server bestimmte Aufgaben innerhalb eines Netzwerks zuweisen. Sie ermöglichen es, Server gezielt für bestimmte Dienste wie Benutzerverwaltung, Dateifreigaben oder Netzwerkdienste einzusetzen.

Zu den wichtigsten Rollen gehört der Active Directory Domain Services (AD DS), der für die zentrale Verwaltung von Benutzern, Gruppen und Computern zuständig ist. Ebenfalls häufig genutzt wird der DNS-Server, der die Namensauflösung im Netzwerk übernimmt.

Weitere zentrale Rollen sind der DHCP-Server zur automatischen Vergabe von IP-Adressen sowie der Datei- und Speicherdienst, der das Bereitstellen von Dateien und Ordnern im Netzwerk ermöglicht.

Rollen werden über den Server-Manager installiert und konfiguriert. Dabei können zusätzlich Features (Funktionen) ergänzt werden, die die Rolle erweitern oder unterstützen.

Durch die gezielte Kombination verschiedener Rollen kann ein Windows Server flexibel an die Anforderungen eines Unternehmens angepasst werden.
